# BioLogist Frontend (React)

## 📌 Description
Frontend for BioLogist e-commerce platform built using React.

## 🚀 Features
- Product listing
- Add to cart
- API integration

## 🛠️ Tech Stack
- React
- Axios / Fetch API

## 🔗 Backend Connection
Connects to Django REST APIs

## ▶️ Run Project
npm install
npm start
